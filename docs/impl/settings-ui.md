## 交互总纲

详见 **[interaction-flow.md](./interaction-flow.md)**（选图 / 启用分离、反馈规范、验收清单）。

## 壁纸页布局（当前）

1. **启用区（顶部）**：`启用壁纸` 开关 + 状态徽章 + 说明  
2. 分屏开关  
3. 分屏列表：每行缩略图 + 选图/移除/（已启用时）更新到桌面  
4. 全局图：大预览 + 选图/移除 +（已启用时）更新到桌面  
5. 缩放（已启用时可就近更新）  
6. 清除全部已选图（会自动停用）

## 关键规范摘要

| 动作 | 桌面是否变化 | 反馈 |
|------|--------------|------|
| 选图 | 否 | 缩略图 +「已选…（尚未启用）」 |
| 打开启用 | 是 | busy →「已启用：模式」 |
| 关闭启用 | 停 overlay | 「已停用（图片仍保留）」 |
| 已启用下「更新到桌面」 | 是 | busy →「已更新到桌面」 |
| 清除全部图 | 停用 | 自动 `disableWallpaperAsync` |

## 模式页 / 菜单栏

- 显式切换模式会 **隐含启用**
- 菜单栏提供：启用 / 更新 / 停用

## 显示器页

- 辅助调整分屏映射；主路径仍在壁纸页
